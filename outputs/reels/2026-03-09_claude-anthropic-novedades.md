## CUERPO — (8 a 55 segundos)

**Segmento 1 — Claude Opus 4.6 y el contexto de 1 millón de tokens (aprox. 12s):**
```
Narración:
"Primero: Claude Opus 4.6 tiene ahora una ventana de contexto de 1 millón de tokens.
Eso significa que puedes darle contratos completos, bases de datos, 200 páginas de
documentos... y no se olvida de nada. Lo que antes tomaba horas de trabajo manual,
ahora lo procesas en minutos."

Indicación visual / CapCut:
Mostrar en pantalla: ilustración o captura de un documento largo siendo procesado.
Texto overlay en la parte inferior: "1 MILLÓN DE TOKENS = más de 750,000 palabras"
Cut rápido al terminar la frase.
```

**Segmento 2 — Adaptive Thinking (aprox. 12s):**
```
Narración:
"Segundo: ahora Claude decide cuánto pensar según la complejidad de lo que le pides.
Para tareas simples va rápido. Para proyectos complejos activa modo máximo.
Es como tener un empleado que sabe cuándo necesita tomarse su tiempo y cuándo
puede responder en segundos. Eso se traduce en menos errores en tus automatizaciones."

Indicación visual / CapCut:
Mostrar diagrama simple de 4 niveles: bajo / medio / alto / máximo.
Texto overlay: "PIENSA SEGÚN LO QUE NECESITAS"
```

**Segmento 3 — Claude Cowork para no-técnicos (aprox. 12s):**
```
Narración:
"Tercero, y esto es lo que más me interesa para ustedes: lanzaron Claude Cowork.
Una herramienta con interfaz visual — sin código — para automatizar tareas de archivos,
documentos y flujos de trabajo completos. No es para developers. Es para gente como
tú y como yo que quiere que la IA trabaje mientras uno hace otra cosa."

Indicación visual / CapCut:
Mostrar captura de pantalla de Claude Cowork (disponible en Anthropic.com).
Texto overlay: "SIN CÓDIGO. SIN PROGRAMAR."
```

**Segmento 4 — El contexto real: lo que esto significa (aprox. 10s):**
```
Narración:
"Y esto ya está pasando en empresas grandes: Goldman Sachs automatizó roles de
```

## Reporte del INVESTIGADOR (base de este guion)

### Fuentes verificadas usadas:

- Claude Opus 4.6 con ventana de 1M tokens (beta) — lanzado febrero 2026
- Adaptive Thinking: 4 niveles de esfuerzo cognitivo — lanzado con Opus 4.6

- Goldman Sachs automatizó roles de contabilidad y compliance con Claude — febrero 2026
- Claude Sonnet 4.6 lanzado el 17 de febrero de 2026 (mismo precio que 4.5)

### Por qué se eligió este ángulo:

El ángulo "novedades que te afectan directamente" conecta mejor con el avatar de Hans
(profesional LATAM que no es técnico pero quiere usar IA para ganar dinero) que un
análisis técnico de benchmarks. Se priorizaron las 3 novedades más accionables y
comprensibles para no-técnicos, dejando afuera datos de benchmarks que no resuenan
con la audiencia.
