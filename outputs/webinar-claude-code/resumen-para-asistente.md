# Resumen de Tareas — Webinar Claude Code
**Para:** Asistente de planificación
**Contexto:** Hans va a hacer un webinar gratuito de 2 días sobre Claude Code para atraer y calentar leads hacia su academia (Momentum AI Academy). Todo se ejecuta mañana martes 25 de marzo.

---

## EL EVENTO

**Qué es:** Masterclass gratuita de 2 días sobre Claude Code (herramienta de IA para automatizar sin programar).
**Para quién:** Abierto al público — seguidores de Instagram, leads que no compraron, y base de datos histórica de leads.
**Objetivo real:** Calentar leads y cerrar ventas de la academia al final del día 2.

| Día | Fecha | Acción |
|-----|-------|--------|
| Preparación | Martes 25 | Grabar, publicar, enviar mensajes, preparar clases |
| Clase 1 | Miércoles 26 | Primera sesión en vivo |
| Clase 2 | Jueves 27 | Segunda sesión en vivo + cierre de ventas |

---

## TAREAS DEL MARTES 25 — EN ORDEN DE PRIORIDAD

### BLOQUE 1 — Contenido y captación (mañana temprano, primer turno)

**0. Crear el grupo de WhatsApp del webinar**
- Crear un grupo de WhatsApp donde se va a centralizar toda la comunicación del evento
- Este link es lo que se envía a todos los que respondan la palabra clave "CLAUDE" en el reel y en las stories
- Sin este link no se puede publicar el reel ni enviar mensajes

**1. Grabar el reel de lanzamiento**
- Reel de 45-55 segundos, talking head directo a cámara
- Anunciar las 2 clases gratuitas sobre Claude Code
- CTA: "Comentá CLAUDE en los comentarios y te mando el link del grupo de WhatsApp"
- Toda la comunicación del evento se maneja por ese grupo
- Guión completo en: `outputs/webinar-claude-code/2026-03-24_webinar-claude-code-reel.md`

**2. Editar el reel en CapCut**
- Cortes cada 4-5 segundos, subtítulos automáticos, textos en pantalla
- CTA final en pantalla: "Comentá CLAUDE"

**3. Publicar el reel en Instagram**
- Caption listo en el mismo archivo del guión
- Compartir también a Stories al publicar
- Fijar en el perfil si es posible

**4. Publicar Stories — Secuencia 1 (4 slides)**
- Anuncio del evento + poll + countdown al miércoles
- Contenido completo en: `outputs/webinar-claude-code/2026-03-24_webinar-claude-code-stories.md`

**5. Enviar mensajes personalizados a leads calientes**
- Leads que tuvieron llamada de ventas y no compraron (15-20 personas)
- Mensaje personal, sin pitch de academia, solo invitar gratis
- Plantillas en: `outputs/webinar-claude-code/2026-03-24_mensajes-leads-masterclass.md`

**6. Enviar blast a base de datos de leads fríos**
- Todos los leads históricos que no cerraron
- Mensaje más neutro, solo oferta de valor gratuito
- Plantillas en el mismo archivo anterior

---

### BLOQUE 2 — Preparación de las clases (tarde, segundo turno)

**7. Ver video de referencia sobre Claude Code (4 horas)**
- Para refrescar el proceso completo de Claude Code de inicio a fin
- Objetivo: tener claridad total antes de enseñarlo en vivo

**8. Planificar el contenido de las dos clases**
- Definir qué se ve el Día 1 (miércoles 26) y qué el Día 2 (jueves 27)
- Preparar la demo en vivo que va a mostrar
- Recomendación del equipo: Día 1 = qué es Claude Code + demo básica; Día 2 = Claude Code + n8n juntos + monetización

**9. Definir la oferta especial para el cierre del Día 2**
- Al final del jueves hay que hacer una transición a la venta de la academia
- Opciones disponibles (Hans debe elegir UNA):
  - Sesión grupal de Q&A exclusiva para asistentes
  - Descuento de $100 en contado (de $997 a $897) válido hasta el viernes 28
  - Primer mes sin cargo en el financiamiento de $100/mes
- Guión de cierre (8 minutos) en: `outputs/webinar-claude-code/estrategia-webinar-claude-code.md`

---

### BLOQUE 3 — Recordatorios y seguimiento (tarde-noche)

- Lo que van a aprender en cada clase
- Archivo: `outputs/webinar-claude-code/2026-03-24_webinar-claude-code-stories.md`
