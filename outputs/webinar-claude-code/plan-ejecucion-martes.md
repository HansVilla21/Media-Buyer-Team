### 7:00 AM — GRABACION DEL REEL (30-40 min)

**Que grabar:** El reel de lanzamiento de la masterclass.
**Archivo de referencia:** `outputs/reels/2026-03-24_webinar-claude-code-reel.md`

**Setup:**
- Camara del telefono frontal, bien iluminado (ventana o luz natural)
- Fondo limpio o tu setup habitual
- No necesitas pantalla ni demo en este reel — es talking head

**Hook elegido (recomendado — usa la Opcion A):**
"Esta semana doy dos clases completamente gratis sobre Claude Code. Y te voy a ser honesto: si no sabes que es Claude Code todavia, estas perdiendo tiempo."

**Lo que dices:**
1. Hook (0-3 seg): version que elegiste
2. Contexto (3-8 seg): "Masterclass dos dias, completamente gratuita. Miercoles 26 y jueves 27 de marzo. Dos sesiones en vivo donde te muestro exactamente como uso Claude Code para automatizar sin ser programador."
3. Cuerpo A (8-20 seg): "Claude Code es una herramienta de Anthropic — le explicas en espanol lo que queres que haga y el escribe el codigo, lo prueba, lo corrige. Vos solo pides el resultado."
4. Cuerpo B (20-32 seg): "Yo lo uso en proyectos reales para clientes. Automatizaciones que antes me tomaban dias enteros, ahora las saco en horas."
5. Cuerpo C (32-40 seg): "En las dos clases te muestro el proceso real. Sin teoria. Pantalla compartida, flujos reales, casos reales."
6. CTA (40-52 seg): "Si queres entrar, comenta CLAUDE abajo. Te mando el link de acceso ahora mismo."

**Como grabar:**
- Una toma completa de calentamiento (no la uses, solo para entrar en confianza)
- Luego graba por bloques si algun segmento no quedo natural
- El hook es lo mas importante: grabalo 3-4 veces con diferentes energias

---

### 7:45 AM — EDICION EN CAPCUT (45-60 min)

**Abri CapCut y hace esto exactamente:**

1. Importa todos los clips que grabaste
2. Arma el orden: hook > contexto > cuerpo A > B > C > CTA
3. Cortes cada 4-5 segundos en talking head — elimina silencios y muletillas
4. Textos en pantalla (blanco, letra grande, mínimo 60pt):
   - Segundo 0: el texto del hook (lo que dices en esos 3 segundos)
   - Cada segmento nuevo: un texto que resume lo que estas diciendo
   - CTA final: "Comentá CLAUDE" en grande, centrado
5. Activa subtitulos automaticos — revisalos rapidamente
6. Musica: opcional, muy baja si la pones
7. Exporta en la maxima calidad disponible

**Texto exacto para el CTA final en pantalla:**
```
Comentá CLAUDE
(mas pequeño abajo): te mando el link gratis
```
