## Notas de Producción

**Formato:** Talking head puro — cámara frontal del teléfono. No necesitás pantalla en este reel.

**Tono que buscás:** Como si le estuvieras contando a un amigo que descubriste algo muy bueno y no querés que se lo pierda. No es hype. Es "mae, en serio, esto es útil — venís o no venís".

**Locación:** Tu setup habitual o donde te sentís más natural. No importa el fondo si el tono es auténtico.

**Cómo grabarlo:**
1. Grabá de corrido una vez completa para calentar.
2. Luego grabá por bloques si algún segmento no quedó natural.
3. El segmento más importante: el hook. Grabá el hook 3-4 veces con diferentes energías y elegís el que se siente más natural, no el más performativo.

**Edición en CapCut:**
- Cortes directos, sin transición. Este reel no necesita efectos.
- Textos en pantalla: blanco, letra grande (mínimo 60pt), aparecen al inicio de cada segmento.
- El primer texto entra en el segundo 0, no después.
- Subtítulos automáticos: activar siempre. Revisar que coincidan.
- Cortes en talking head: cada 4–5 segundos. Eliminar silencios y muletillas.
- Música: opcional, muy baja si la ponés. El contenido habla solo.
