## Métricas que Importan (en orden de prioridad)

### Tier 1 — Señales de Profundidad
| Métrica | Qué indica | Benchmark |
|---------|-----------|-----------|
| **Comentarios sustanciosos** | Resonancia real del contenido | >3 por post |
| **Profile views** | Curiosidad generada sobre Hans | Tendencia creciente semana a semana |
| **Saves / Bookmarks** | Contenido con valor de referencia | >5% del engagement total |
| **Shares / Reposts** | Amplificación orgánica real | Cualquier share es señal fuerte |
| **DMs recibidos** | Interés personal / señal de lead | Documentar cada uno |

### Tier 2 — Señales de Crecimiento
| Métrica | Qué indica | Benchmark |
|---------|-----------|-----------|
| **Follower growth** | Métrica compuesta de todo lo demás | >1%/semana en fundación |
| **Impresiones** | Alcance del contenido | Tendencia, no absoluto |
| **Engagement rate** | Ratio engagement/impresiones | 4-8% nicho técnico LATAM |

### Tier 3 — Vanity (monitorear pero no optimizar)
| Métrica | Por qué es vanity |
|---------|------------------|
| **Likes / Reactions** | Fáciles de dar, no indican interés profundo |
| **Connection requests** | Pueden ser bots o spam |

| Formato | Engagement mediano | Notas |
|---------|-------------------|-------|
| Carrusel PDF | 21.77% | Formato rey — 3x video, 4x texto |
| Video nativo | 7.35% | Segundo lugar — siempre nativo, nunca link |
| Imagen/Infografía | 6.52% | Buena para datos y estadísticas |
| Link post | 3.81% | Penalizado — link siempre en comentario |
| Texto puro | 3.18% | Funciona con hook fuerte y voz auténtica |

Engagement rate general LinkedIn 2025: **6.2%** (la más alta de todas las redes sociales).

## Framework de Decisión Post-Retro

Después de analizar cada post de la semana:

| Señal | Decisión | Acción |
|-------|---------|--------|
| >2x engagement promedio + comentarios del avatar | **DOBLAR** | Hacer más de este tipo/formato/ángulo |
| Engagement promedio + 0 comentarios del avatar | **ITERAR** | Mismo formato, diferente ángulo/hook |
| <0.5x engagement promedio | **ANALIZAR** | ¿Hook? ¿Formato? ¿Horario? ¿Tema? |
| Muchas impresiones + bajo engagement | **HOOK ISSUE** | Tema tiene interés, ejecución no enganchó |
| Pocas impresiones + alto engagement | **DISTRIBUCIÓN** | Contenido bueno, timing o golden hour falló |
| Poll >100 respuestas | **OPORTUNIDAD** | Hacer follow-up post con resultados |
| Post que genera DMs / profile views altos | **SEÑAL LEAD** | Documentar ángulo que generó esto |

| Nivel | Score | Acción |
