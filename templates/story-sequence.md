# Template: Secuencia de Stories

> Copia este template para cada secuencia nueva.
> Guarda el archivo como: `outputs/stories/YYYY-MM-DD_[tema-corto].md`
> Una secuencia = 2 a 6 stories encadenadas con un objetivo específico.

---

## Metadata

```
Título: [Tema de la secuencia]
Fecha de creación: YYYY-MM-DD
Fecha de publicación objetivo: YYYY-MM-DD
Número de slides: [2–6]
Objetivo: [ ] Educación  [ ] Venta  [ ] Engagement  [ ] BTS  [ ] Urgencia  [ ] Testimonio
Estado: [ ] Borrador  [ ] Listo  [ ] Publicado
```
